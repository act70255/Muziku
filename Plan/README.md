# AI YouTube Music Channel Plan

這份文件集用來規劃一個以 `AI 生成 Lo-fi / Study / Chill 音樂` 為核心、面向全球混合市場、以 `YouTube 廣告分潤` 為主要變現方式的頻道。

## 文件導覽

1. [plan.md](./plan.md)
整體總表、閱讀入口、核心決策摘要。

2. [plan-business.md](./plan-business.md)
商業與流量規劃，包含受眾、內容策略、KPI 與營利節奏。

3. [plan-technical.md](./plan-technical.md)
技術與製作流程規劃，包含 AI 音樂生成、影片組裝、上傳流程與自動化。

4. [plan-startup.md](./plan-startup.md)
90 天執行規劃、資源配置、風險管理與營運演進。

5. [AI-music-tools.md](./AI-music-tools.md)
主流 AI 音樂服務比較、授權重點、YouTube 適配度與選型建議。

## 建議閱讀順序

1. 先看 [plan.md](./plan.md) 了解總體方向
2. 再看 [plan-business.md](./plan-business.md) 確認頻道策略
3. 接著看 [plan-technical.md](./plan-technical.md) 確認產製流程
4. 再看 [AI-music-tools.md](./AI-music-tools.md) 確認工具選型
5. 最後看 [plan-startup.md](./plan-startup.md) 安排 90 天執行節奏

## 專案定位摘要

- 主題：Lo-fi / Study / Chill
- 市場：全球混合市場
- 內容型態：30 分鐘到 3 小時長影片
- 音樂來源：純 AI 生成音樂
- 主要變現：YouTube 廣告分潤
- 規劃週期：90 天成長規劃

## 使用方式

這是一套策略與執行規劃文件，不是最終 SOP 定稿。建議先依照 `plan.md` 做方向判斷，再逐步把子檔案內容轉成實際待辦、預算表、工具清單與每週追蹤表。
